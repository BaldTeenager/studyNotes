### docker命令

```
创建一个新的容器并运行一个命令
docker run 

查看已经创建的容器
docker ps -a

查看正在运行的容器
docker ps -s 

启动容器
docker start e84ab1edd3fb 

进入msql容器
sudo docker exec -it e84ab1edd3fb /bin/bash

在宿主机上复制容器内文件
docker cp e84ab1edd3fb:/test.sql /home

退出容器但是不关闭容器
Ctrl+P+Q

根据容器id查看pid
sudo docker inspect -f '{{.State.Pid}}' c4f67fa13989


```
